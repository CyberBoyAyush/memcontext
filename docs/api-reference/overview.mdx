## Base URL

```text
https://api.memcontext.in
```

## Authentication

Send an API key in the `X-API-Key` header.

## Main endpoint groups

- save and search memory
- list and manage memory records
- inspect memory history
- forget or delete memories
- submit retrieval feedback
- fetch pre-aggregated profile context

## OpenAPI reference

This docs site includes an OpenAPI spec in `openapi.yaml` so Mintlify can generate interactive API pages and code examples.
