---
title: "save_memory"
description: "Persist durable user or project information through MCP."
---

## Purpose

Use `save_memory` when information is durable and worth remembering across sessions.

## Parameters

- `content`
- `category`
- `project`
- `validUntil`

## Guidance

- save preferences, facts, decisions, and important project context
- do not save one-off debug noise
- use `validUntil` only for time-sensitive memory
